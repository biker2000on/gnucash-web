## Vision

### 10x Check
The 10x version is a full portfolio management system where the app auto-imports brokerage data, auto-classifies transactions, auto-assigns lots optimally for tax efficiency, and surfaces actionable tax planning — all self-hosted. This plan (lot assignment + tax harvesting) is the critical step from "read-only lot viewer" to "smart automation."

## Accepted Scope (added to this plan)
- Cross-account wash sale detection (all STOCK/MUTUAL accounts in book)
- Configurable ST/LT tax rates with projected savings calculation
- Average method creates separate lots per buy, displays averaged cost
- "Clear All Lot Assignments" button for bulk undo
- Unlinked splits count + quick-assign in LotViewer
- No lot splitting (match GnuCash desktop behavior)

## Architecture Decisions (from CEO Review)
- **Extracted components:** LotAssignmentPopover and AutoAssignDialog are standalone components, not embedded in AccountLedger.tsx
- **Database transactions:** All bulk operations (auto-assign, clear-assign) wrapped in `prisma.$transaction()` for atomicity
- **Security validation:** Split-to-lot assignment validates both belong to same account
- **Sell allocation:** Sells assigned to primary FIFO/LIFO lot even if shares go negative (matches GnuCash desktop)
- **Schema change:** Add `lot_assignment_method` column to `gnucash_web_account_preferences` (additive, zero-downtime)
